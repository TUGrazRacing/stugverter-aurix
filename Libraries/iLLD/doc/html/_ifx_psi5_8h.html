◆ IfxPsi5_FractionalDividerRegister

enum IfxPsi5_FractionalDividerRegister

Options for choosing different Fractional Divider Registers.

Enumerator
IfxPsi5_FractionalDividerRegister_normal 

The Fractional Divider Register controls the input clock f_fracdiv.

IfxPsi5_FractionalDividerRegister_lowbitrate 

The Fractional Divider Register for lower Bit Rate contains the pre-divider that defines the time resolution of the f_125.It divides f_fracdiv by a factor and provides f_125 to all channels.

IfxPsi5_FractionalDividerRegister_highbitrate 

The Fractional Divider Register for Higher Bit Rate contains the pre-divider that defines the time resolution of f_189. It divides f_fracdiv by a factor and provides f_189 to all channels.

IfxPsi5_FractionalDividerRegister_timestamp 

The PSI5 Module Time Stamp Predivider Register contains the pre-divider that defines the time resolution of the Time Stamp Registers TSRA/B/C and the Sync Pulse Time Base Counter SBC. It divides fPSI5 by a factor. It contains as well the bits for reset control of the time stamp counters.

◆ IfxPsi5_ReceiverControlRegister

enum IfxPsi5_ReceiverControlRegister

Optiond for different Receive Control Registers(RCRA,RCRB,RCRC)

Enumerator
IfxPsi5_ReceiverControlRegister_a 

Receiver Control Register A configures the payload length of the up to 6 frames in the up to 6 slots.

IfxPsi5_ReceiverControlRegister_b 

Receiver Control Register B configures up to 6 frame types in the up to 6 slots.

IfxPsi5_ReceiverControlRegister_c 

Receiver Control Register C configures bit rate and time stamp sources.
