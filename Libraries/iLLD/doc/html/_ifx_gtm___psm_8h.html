This Module provides the standard interfaces to manipulate and read PSM sub-module registers in GTM. This Module can be used to abstract the register names and addressing from higher layer software code. PSM includes: FIFO, A2F (AEI to FIFO) and F2A (FIFO to ARU) Interfaces.
