◆ IfxSmuStdby_enableAlarmStatusClear()

IFX_INLINE void IfxSmuStdby_enableAlarmStatusClear ( void )

This function enable / disables he Alarm status clear in the Smu Standby command register . This controls if a status flag set in an AGx register upon detection of the alarm event can be cleared by software or not. When ASCE is enabled, software shall write a 1 to bit position in AGx to clear the bit (W1C). When a W1C action takes place the ASCE bit is automatically cleared to 0 by hardware and software shall set the ASCE bit again.
