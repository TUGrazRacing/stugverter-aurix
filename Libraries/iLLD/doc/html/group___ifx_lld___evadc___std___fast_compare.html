◆ IfxEvadc_BoundaryFlagActivationMode

enum IfxEvadc_BoundaryFlagActivationMode

Selects the behaviour of BoundaryFlag in FCBFL.B.BFA.

Enumerator
IfxEvadc_BoundaryFlagActivationMode_0 

Set boundary flag BFL if result FCR = 1 (input above the defined band or compare value), clear if FCR = 0 (input below)

IfxEvadc_BoundaryFlagActivationMode_1 

Set boundary flag BFL if result FCR = 0 (input below the defined band or compare value), clear if FCR = 1 (input above)
