IFX_INLINE void IfxPmsPm_startLoadJumpTimer (void)
 Starts Load jump timer. The SCU_PMTRCSR0.B.LJTSTRT bitfield remains set on a write and is cleared when SCU_PMTRCSR2.B.LJTOV bit is set if SCU_PMTRCSR0.B.LJTOVEN bit is enabled(Update of Load Jump Timer Overflow Flag SCU_PMTRCSR2.B.LJTOV is enabled ).
 
IFX_INLINE void IfxPmsPm_stopLoadJumpTimer (void)
 This stops the Load jump timer. The Load Jump Timer stops counter at the current value and timer re-starts from that value when LJTSTP is cleared and LJTSTRT is set.
