IFX_INLINE void IfxFlash_userContentCount (uint32 wordAddr)
 The function issues command sequence for User Content count.The result of this function is dependent on the difference of the number of logic 1 bits in the selected pages at the erase-verify condition (N_dv) and at the selected control gate voltage (N_pv). Calling this function with different control gate voltages allows calculation of the Vth distribution.
 
IFX_INLINE void IfxFlash_userMarginCount (uint32 wordAddr)
 The function issues command sequence for User Margin Count. The result of this function is the number of logic 1 bits in the selected pages at the selected reference current. Calling this function with different reference currents allows calculation of the cell current distribution.

IFX_INLINE void IfxFlash_writeBurstOnce (uint32 pageAddr)
 Performs the "Write Burst once" sequence. The command starts the programming process for an aligned group of pages as the normal "Write Burst" does. But before programming it checks if the pages are erased. If the page is not erased (allowing correctable errors) the command fails with PVER and EVER. The command is only supported for PFlash. On sectors with "write-once" protection only this write command can be applied.
