How to use the GTM ATOM PWM Driver
GTM » Atom Interface Drivers » ATOM PWM HL Interface Driver

This driver implements the PWM functionalities as defined by library_srvsw_stdif_pwmhl. The user is free to use either the driver specific APIs below or to used the standard interface APIs.

Specific implementation

Implementation is similar to TOM PWM HL Interface Driver

For a detailed configuration of the microcontroller, see IfxGtm_AtomDtm_PwmHl_init().

Usage example

Initialisation is done by, e.g:

IfxGtm_AtomDtm_PwmHl_Config driverConfig;
IfxGtm_AtomDtm_PwmHl driverData;
IfxStdIf_PwmHl pwmhl;
IfxGtm_AtomDtm_PwmHl_initConfig(&driverConfig, &MODULE_GTM);
IfxGtm_AtomDtm_PwmHl_init(&driverData, &driverConfig);
IfxGtm_AtomDtm_PwmHl_stdIfPwmHlInit(pwmhl, &driverData);
IfxGtm_Atom_Dtm_PwmHl_s
GTM ATOM PWM driver.
Definition IfxGtm_Atom_Dtm_PwmHl.h:163

During run-time, the interface functions shall be used, e.g.:

IfxStdIf_Timer* timer = IfxStdIf_PwmHl_getTimer(pwmhl);
Ifx_TimerValue onTime[3]; // assume configured for three HL channels
onTime[0] = 10;
onTime[1] = 20;
onTime[2] = 30;
IfxStdIf_Timer_disableUpdate(timer);
IfxStdIf_Timer_setPeriod(timer, period);
IfxStdIf_PwmHl_setOnTime(pwmhl, onTime);
IfxStdIf_Timer_applyUpdate(timer);
